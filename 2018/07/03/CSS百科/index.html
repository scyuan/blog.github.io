在实际项目中，会遇到各种各样的css问题，这其中有以前没用过的，或者一些兼容问题等等。以前都是记录在各个地方，不好总结和查漏补缺。

1. transform属性引起的抖动

在一个个人网站的项目中，当hover一个元素，某个元素的transform:scale(1)->transform:scale(20)。但是会引起其他元素的抖动。

原因：因为transform重绘引起的？

解决：transform:translateZ(0);

3. position：fixed失效问题

position:fixed是基于视窗布局的，与文档位置没有关系。但是假如父容器有transform属性，position：fixed就会失效

解决：将fixed布局的盒子移出含有transform属性的盒子。
